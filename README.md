## API Endpoints

- `POST /expenses` - Create a new expense
- `GET /expenses` - Get all expenses
- `GET /expenses/:id` - Get a specific expense
- `PUT /expenses/:id` - Update an expense
- `DELETE /expenses/:id` - Delete an expense
